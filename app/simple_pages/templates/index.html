Project Outline

Building on top of our past projects, this capstone web app ties together all the technologies covered in class. The objectives were to implement Docker, Flask, Bootstrap, SQLite and Python and serve a web app on Heroku that allows users to create accounts, upload and upload/view sample CSV formatted financial statements. Critical backend pieces of this project was building out the logging functions and the automated testing.
