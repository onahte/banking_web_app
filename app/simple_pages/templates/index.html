Project Outline

Building on top of our past projects, this capstone web app ties together all the technologies covered in class. The objectives were to implement Docker, Flask, Bootstrap, SQLAlchemy and Python and serve a containerized web app on Heroku that allows users to create accounts and upload/view sample CSV formatted financial statements. Critical backend pieces of this project were building out the logging functions and the automated testing. And as with our previous projects, Github actions were used to build our automated CI/CD pipeline.
